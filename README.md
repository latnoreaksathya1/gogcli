## üìñ Usage Instructions

Here are some basic commands you can use in gogcli:

1. **Check your email:**
   ```
   gogcli gmail check
   ```

2. **Schedule an event:**
   ```
   gogcli gcal add --date "2023-10-14" --event "Meeting with Team"
   ```

3. **Upload a file to Drive:**
   ```
   gogcli gdrive upload --file "path/to/your/file.txt"
   ```

4. **List your contacts:**
   ```
   gogcli gcontacts list
   ```

Each command will provide necessary prompts and help along the way.
